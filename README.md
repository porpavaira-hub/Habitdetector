# 🧠 Habit Detector API

A simple backend project built using FastAPI to track habits, calculate streaks, and detect missed days.

---

## 🚀 Features
- Add daily habits
- Track habit streaks
- Detect habit breaks (missed days)
- SQLite database storage

---

## ⚙️ Tech Stack
- Python
- FastAPI
- SQLite
- SQLAlchemy
